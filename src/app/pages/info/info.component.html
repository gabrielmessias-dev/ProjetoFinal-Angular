DiagnoTec X: Inovação e Cuidado Digital

A DiagnoTec X não é apenas uma plataforma digital; ela representa o ponto alto de um ciclo de aprendizado intensivo e a concretização de um objetivo acadêmico significativo.

Um Projeto de Conclusão para a Ford Enter

Este projeto é o Trabalho de Conclusão de Curso (TCC) do curso profissionalizante Ford Enter, focado no Desenvolvimento Front-End. A DiagnoTec X foi concebida como uma empresa fictícia com o propósito de servir como o cenário prático e demonstrativo de todo o conhecimento e habilidades adquiridos ao longo desta formação. É a materialização de um desafio que visa aplicar tecnologias de ponta na criação de soluções web robustas e intuitivas.

Meu empenho neste projeto final reflete o compromisso com a excelência e a paixão por construir interfaces que não apenas funcionam, mas que também oferecem uma experiência de usuário superior, especialmente em um setor tão sensível e crucial quanto o da saúde.

Tecnologias e Ferramentas Utilizadas

A construção da DiagnoTec X foi realizada utilizando um ecossistema de tecnologias e ferramentas modernas, amplamente empregadas no mercado de desenvolvimento web, garantindo a solidez, escalabilidade e manutenibilidade do projeto:

  • Angular (v19.x): O framework principal para a construção da Single Page Application (SPA), aproveitando seus recursos para componentes, roteamento e gerenciamento de estado.
  • TypeScript: A linguagem de programação utilizada, que estende o JavaScript adicionando tipagem estática, resultando em código mais robusto, legível e com menos erros em tempo de desenvolvimento.
  • Bootstrap 5: Framework CSS para um design responsivo, moderno e acelerado, garantindo uma interface adaptável a diferentes dispositivos.
  • RxJS: Biblioteca para programação reativa, essencial para o gerenciamento de eventos assíncronos e fluxos de dados.
  • Angular CLI: Ferramenta de linha de comando que agiliza o desenvolvimento Angular, desde a criação de projetos e componentes até o build e testes.
  • Firebase Authentication: Serviço de backend online para gerenciamento de usuários, provendo autenticação segura via e-mail e senha.
  • Firebase Firestore: Banco de dados NoSQL baseado em nuvem, em tempo real, utilizado para a persistência e gerenciamento de dados dos exames (agendamentos e histórico).
  • Firebase Hosting: Serviço de hospedagem para disponibilizar a aplicação web online de forma rápida e segura.
  • VS Code: O ambiente de desenvolvimento integrado (IDE) principal para a codificação do projeto.
  • Git e GitHub: Utilizados para controle de versão, colaboração e implantação.

Durante a fase de desenvolvimento e prototipagem local, ferramentas como JSON Server foram empregadas para simular um backend inicial e Local Storage para persistência temporária de dados no navegador.

Nosso Compromisso (como projeto)

A DiagnoTec X não é apenas a demonstração de um conjunto de tecnologias, mas a prova da capacidade de traduzir requisitos complexos em uma solução funcional e amigável. Ela exemplifica a aplicação de boas práticas de desenvolvimento front-end, com foco na experiência do usuário, modularidade do código e clareza da interface. Este é o futuro da interação digital na saúde.
